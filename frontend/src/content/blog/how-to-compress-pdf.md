# 如何压缩 PDF 文件？5种方法对比

PDF 文件因其格式固定、跨平台兼容的特点，成为文档分享的首选格式。但随着内容增多，PDF 文件体积可能变得很大，给存储和传输带来不便。本文将介绍 5 种压缩 PDF 的方法，帮助你选择最适合的方案。

## 为什么需要压缩 PDF？

### 常见问题

- **邮件附件限制**：大多数邮箱限制附件大小为 25MB
- **存储空间不足**：大量 PDF 占用过多硬盘空间
- **上传速度慢**：大文件上传耗时，网络不稳定时容易失败
- **移动端受限**：手机流量有限，大文件下载成本高
- **网站上传限制**：许多平台限制上传文件大小

### 压缩的好处

- ✅ 减小文件体积，节省存储空间
- ✅ 加快上传下载速度
- ✅ 满足邮件和平台的文件大小限制
- ✅ 降低带宽消耗，节省流量
- ✅ 便于在移动设备上查看和分享

## 方法 3：Microsoft Word（PDF 转 Word 再转 PDF）

### 适用场景

- 已安装 Microsoft Office
- PDF 内容以文字为主
- 不需要保留复杂格式

### 使用方法

## 方法 4：命令行工具（技术用户）

### 适用场景

- 开发者或技术用户
- 需要批量处理
- 需要自动化脚本
- 服务器环境

### 常用工具

#### Ghostscript

```bash
# Linux/Mac
gs -sDEVICE=pdfwrite -dCompatibilityLevel=1.4 -dPDFSETTINGS=/ebook -dNOPAUSE -dQUIET -dBATCH -sOutputFile=output.pdf input.pdf

# 压缩级别选项：
# /screen   - 最低质量，最小文件（72dpi）
# /ebook    - 较低质量（150dpi）
# /printer  - 打印质量（300dpi）
# /prepress - 出版质量（300dpi+）
# /default  - 默认设置
```

#### qpdf

```bash
# 线性化（优化 Web 查看）
qpdf --linearize input.pdf output.pdf

# 压缩
qpdf --compress-streams=y input.pdf output.pdf
```

### 优点

- ✅ 完全免费
- ✅ 批量处理能力强
- ✅ 可集成到脚本
- ✅ 服务器友好

### 缺点

- ❌ 需要技术知识
- ❌ 无图形界面
- ❌ 学习成本高

## 压缩技巧与建议

### 选择合适的压缩级别

1. **屏幕查看**：选择高压缩率，文件最小
2. **打印**：选择中等压缩，平衡质量和大小
3. **出版**：选择低压缩或无损，保持最高质量

### 优化 PDF 内容

1. **压缩图片**
   - 使用适当分辨率的图片
   - 避免使用 BMP 格式
   - 压缩后再插入 PDF

2. **减少字体**
   - 使用常见字体（Arial、Times New Roman）
   - 避免嵌入过多字体
   - 子集化字体（只嵌入使用的字符）

3. **删除不必要内容**
   - 移除隐藏图层
   - 删除注释和标记
   - 清理元数据

### 批量压缩

如果需要处理大量 PDF，建议使用：
- Adobe Acrobat 的动作向导
- 命令行脚本自动化
- 专业的批量处理工具

## 常见问题

### 压缩后质量会变差吗？

取决于压缩级别：
- **轻度压缩**：几乎看不出差别
- **中度压缩**：文字清晰，图片略有损失
- **高度压缩**：可能出现模糊，适合屏幕查看

### 压缩能减小多少体积？

- 文字为主：10-30%
- 图文混合：30-70%
- 图片为主：50-90%
- 扫描版 PDF：60-95%

### 压缩后的 PDF 能恢复吗？

不能。压缩是单向过程，建议：
- 保留原始文件
- 压缩前备份
- 确认压缩效果满意后再删除原文件
