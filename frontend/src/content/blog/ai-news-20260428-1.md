# Google Gemini 1.5 Pro 全面开放：100万Token上下文重塑文档处理

## 新闻概述

2026年4月，Google 正式宣布 Gemini 1.5 Pro API 全面开放，这一里程碑式的发布标志着大语言模型进入"超长上下文"时代。Gemini 1.5 Pro 最引人注目的特性是其惊人的 100万 Token 上下文窗口，相当于可以一次性处理约 700,000 个汉字或 1500 页标准文档。

对于文档处理领域而言，这意味着革命性的变化：用户现在可以将整本书、完整的代码库、或者数千页的法律文档一次性输入给 AI，让它进行深度分析、总结和问答。这彻底改变了我们处理长文档的方式。

**适用场景**：
- 个人用户试用
- 小规模文档分析
- 学习和研究

### 方法 2：API 集成

**Python 示例**：
```python
import google.generativeai as genai

# 配置 API
 genai.configure(api_key="YOUR_API_KEY")

# 初始化模型
model = genai.GenerativeModel('gemini-1.5-pro')

# 读取 PDF 文件
with open("document.pdf", "rb") as f:
    pdf_content = f.read()

# 发送请求
response = model.generate_content([
    {
        "mime_type": "application/pdf",
        "data": pdf_content
    },
    "请分析这份文档，提取关键信息并生成摘要"
])

print(response.text)
```

**适用场景**：
- 企业级应用
- 自动化流程
- 大规模文档处理
